# FreshShare App

FreshShare is a simple Flask-based web app that helps ISU students share safe, packaged, temperature-controlled food. The app allows users to post food, verify safety, browse available items, view campus events with free food, find free food organizations, and leave feedback.

## Features
- Post packaged food with details and dietary filters  
- Verify, accept, or decline food posts  
- Browse, order, or cancel food availability  
- Campus events with food provided
